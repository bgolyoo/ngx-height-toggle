ngx-height-toggle

Angular(2+) directive to change the height of html elements to their content's height on toggle and on content change.

Dependencies

  • Angular (tested with 6.0.5)
  • Angular CLI (tested with 6.0.8)

Installation

After installing the above dependencies, install `ngx-height-toggle` via npm:
